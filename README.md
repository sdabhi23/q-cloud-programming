# Q Cloud Programming

This repo contains implementations of some basic quantum circuits using Qiskit and Cirq

## Dependencies

* Cirq (v0.7.0)
* Qiskit (v0.18.0)

## Algorithms Implemented:

* Random bit generator
* Half adder
* Quantum teleportation

  Reference: https://github.com/quantumlib/Cirq/blob/master/examples/quantum_teleportation.py
